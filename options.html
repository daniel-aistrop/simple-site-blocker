How patterns work

Simple patterns match by substring. If you enter helloWorld.com, it will block any URL that contains that text (e.g. https://helloWorld.com, https://www.helloWorld.com/‚Ä¶).

If you enter just helloWorld, it will block any URL whose host contains that text (e.g. helloWorld.com, myhelloWorld.net).

Regex patterns start with re: and let you write full regular expressions, for example: re:^https?://(www\\.)?helloWorld\\.com/.*$.
